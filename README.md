<h1 align="center">Erymont</h1>

<p align="center">
  A modern Linux container management CLI.
</p>

---


## ✨ Features

- 🚀 Fast and lightweight
- 🎨 Beautiful CLI interface
- 📦 Install Linux distributions
- 💻 Interactive shell
- 💾 Backup & Restore
- 🩺 Built-in diagnostics
- ⚙️ Easy configuration
  

---

## 📋 Requirements

- Linux
- Bash 5.0+
- curl or wget
- tar
- gzip

---

## 📥 Installation

Clone the repository:

```bash
git clone https://github.com/Erymont/Erymont.git
```

Go to the project:

```bash
cd Erymont
```

Make it executable:

```bash
chmod +x ery
```

Run Erymont:

```bash
./ery help
```

---

## 🚀 Quick Start

Install Ubuntu:

```bash
./ery install ubuntu
```

Start Ubuntu:

```bash
./ery start ubuntu
```

Open a shell:

```bash
./ery shell ubuntu
```

---

## 📚 Commands

| Command | Description |
|---------|-------------|
| `ery install ubuntu` | Install Ubuntu |
| `ery start ubuntu` | Start Ubuntu |
| `ery stop ubuntu` | Stop Ubuntu |
| `ery restart ubuntu` | Restart Ubuntu |
| `ery shell ubuntu` | Open shell |
| `ery backup ubuntu` | Create backup |
| `ery restore ubuntu` | Restore backup |
| `ery list` | List installed environments |
| `ery doctor` | Check system health |
| `ery config` | Open configuration |
| `ery update` | Update Erymont |
| `ery version` | Show version |
